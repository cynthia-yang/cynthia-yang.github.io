有趣的访谈方法

有人说定性研究方法是艺术和科学的结合，由于在市场调研中，定性方法通常被作为探索性研究方法，用于发现机遇、激发创意，因此，它可以更加灵活，更具创造性。在市场调研实践中，研究人员广泛吸取各学科各领域的智慧，开发出了许多新颖有趣的访谈方法，跳出了焦点小组和深度访谈的桎梏。今天，我们向大家介绍的“投射法”就是脱胎于临床心理学用于心理治疗的投射测验。

所谓“投射测验”（projective test），就是给被试呈现一个模棱两可的刺激，并假设被试会把自己的无意识过程投射到上面。这种方法假设，人对情境的解释依赖从情境中获得的线索，如果不存在显而易见的线索，则人们只好根据与自身有关的情绪、态度、信念进行解释。人们相信，这种方式能够在一定程度上揭示出深层的心理状态。

临床心理学中最著名且应用最广泛的两个投射测验是罗夏墨迹测验和主题统觉测验（TAT，Thematic Apperception Test）。罗夏墨迹测验是让被试观看一些随机墨迹图形，让被试观看并想象“这是什么？”。

今天开始正式向大家介绍具体的调研方法。我们先从定性调研开始。首先，将向大家介绍应用最为广泛的一种：焦点小组访谈。这种方法应用是如此广泛，以至于有些市场调研机构把焦点小组访谈等同于全部定性调研方法。

焦点小组访谈又叫“座谈会”，就是在一名经过训练的小组主持人的主持下，以一种自然的方式对少数调查对象进行组群式访问。相信大家对这种方法多少都有些了解，但很多人的了解都是非常模糊的，事实上，焦点小组的开展并不简单，有以下几个重要的方面要向大家介绍。

首先，关于焦点小组的人员构成。焦点小组访谈要根据一定的相关特征来筛选参与者，我们建议，针对一个问题可以开展系列焦点小组访谈，小组内部成员要保持高度的同质性，而组间要保持异质性。这样，就能够确保变量之间不会相互干扰，而通过组间差异又能很好地观察不同变量的影响。

案例：Dior公司希望重新推出老牌香水——Realities，该品牌在2010年退出市场。Dior公司计划在目标顾客中开展焦点小组访问并让他们自由表达对目前香水品牌使用情况、满意度，并对老牌香水（包括Realities）的记忆。Dior公司决定在25~49岁的女性人群中开展焦点访问。

确定了研究问题，下一步就是研究方案的设计。研究方案需要根据研究的类型和目标来选择合适的研究方法。

首先，我们需要区别探索性调研、描述性调研和因果性调研。探索性调研可以帮助更好地确定一个营销问题或识别一个市场机遇，澄清模糊的态势或者发现有可能成为潜在商业机遇的创意。描述性调研则是在情境相对清晰之后，对所面临的问题进行全面的描绘，因果性调研则更进一步，针对问题提出一个明确的假设然后对其进行验证。我们举例说明三种类型的调研：

探索性调研的例子：“我们的销量持续下滑，却找不到明确地原因”“快餐消费者感兴趣的是哪类产品？”
描述性调研的例子：“与我们的主要竞争对手相比，经常光顾我们商店的是哪类人群？”“对于我们的顾客来说，最为重要的产品特性是什么？”
因果性调研的例子：“采用蓝色包装消费者会增加购买量”“这两种广告方案哪种会更有效？”

总之，探索性调研面对的决策情境是高度模糊的，其目标是提供对问题的看法与理解，研究过程灵活、非结构化，调研结果不能直接应用到管理决策中，需要进一步的调研紧随其后进行验证。描述性调研面对的决策情境是部分确定的，通过调研提供更多信息以提出假设；因果调研则是对情境完全清晰，其目标是验证提出的假设，研究过程是正式且结构化的，结果可以直接作为管理决策的依据。

只有准确界定了市场调研问题，后面的调研设计和调研实施才有意义，因此，在市场调研过程中，最重要的一步就是正确界定市场调研问题。然而，有数据显示，90%在中国进行的调研都因模糊的问题界定而导致研究效率低下。作为第一篇推送的文章，小编今天就向大家介绍一下如何正确界定市场调研的问题。

我们先从几个case开始。在界定研究问题时，经常发生的情况是：征兆混淆了事实。单纯根据征兆界定问题，可能会错误地关注了表面的、虚假的联系，忽略了深层的、真正的原因，将研究导向错误的方向。

Case1:
征兆：某城市一个历时20年之久的游泳协会，会员人数连年下降；但城中新建的带有波浪池和水上滑梯的大型水上公园却人流旺盛。
根据征兆确定的问题：附近居民更喜欢广阔的水上公园，对游泳池没有什么兴趣。
真正的问题：城市中人口结构发生变化，孩子们已经长大，上了年纪的居民不再游泳。
